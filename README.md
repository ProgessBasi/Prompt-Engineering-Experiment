# Prompt Engineering Experiment

This project evaluates different Large Language Models (LLMs) on the GSM8K mathematical reasoning dataset using the Groq API in Google Colab.

## Features

- GSM8K dataset evaluation
- Multi-model testing
- Prompt engineering experiments
- Parallel API key support
- Automatic progress saving
- Google Colab compatible

---

## Models Used

- `llama-3.1-8b-instant`
- `meta-llama/llama-4-scout-17b-16e-instruct`

---

## Installation

Install required libraries:

```bash
pip install groq datasets
```

---

## How to Run

1. Open the notebook in Google Colab
2. Add Groq API keys in Colab Secrets
3. Mount Google Drive
4. Run all notebook cells
